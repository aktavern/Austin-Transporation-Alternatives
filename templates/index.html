About This Project

The Austin Micromobility data set contains information about the usage of scooters and bicycles within the city of Austin. This app reviews the data, and demonstrates a machine learning model which can predict the range of demand for either bikes or scooters on a given day at a given time with 90% accuracy.

Data Analysis

Take a deep dive into the underlying trends behind the data. Analysis includes the total number devices, the number of miles and hours the micromobility devices have been used, and an analysis on usage patterns.
